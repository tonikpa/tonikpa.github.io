YOU, THE MEMORIOUS

You, the Memorious is a web experience made in a weekend. This website has recorded every memory you have ever had, although prolonged use of its service may poison the memory pool. This project is heavily inspired by the work of Jorge Luis Borges. It uses procedural generation to generate memories that are initially amusing but that ultimately reveal the trappings of nostalgia and obsessive record keeping.

In this project I did all parts of the development.
