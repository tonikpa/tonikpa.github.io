THE REVERSED QUEEN OF WANDS

The Reversed Queen of Wands is a game made in a weekend. With this project I explored creating an oppressive lonely environment and using sounds in interesting ways. This game is about loneliness, about shouting into the void, about attempting to connect and failing every time.

In this project I did most parts of the development, except I borrowed a bunch of sfx and textures.
